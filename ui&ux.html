UI/UX Design Exam

1. What does UX stand for?

  • User Experience
  • User Execution
  • Unified Experience
  • User Exchange

2. What is the main goal of UI design?

  • To plan the user journey
  • To make interfaces visually appealing and easy to use
  • To create wireframes only
  • To test user behavior

3. What is a wireframe?

  • A finalized UI layout
  • A color scheme guide
  • A prototype with animations
  • A basic layout sketch of a page

4. Which tool is commonly used for UI/UX design?

  • Figma
  • Eclipse
  • Visual Studio Code
  • Notepad++

5. What is the purpose of user personas?

  • To define color schemes
  • To describe layouts
  • To represent different user types and needs
  • To test font styles

6. What is the “Hick’s Law” in UX design?

  • Users prefer complex interfaces
  • The more options, the longer the decision-making time
  • Simpler layouts are unattractive
  • Colors affect emotions

7. Which color is often associated with trust in design?

  • Blue
  • Red
  • Yellow
  • Black

8. What does “responsive design” mean?

  • Fast animations
  • Multi-color UI
  • Only desktop-friendly layout
  • Layout adapts to all screen sizes

9. What does a prototype represent?

  • A list of requirements
  • A working model of a design concept
  • A final product
  • A site map

10. Which principle focuses on keeping designs simple and essential?

  • Contrast
  • Consistency
  • Minimalism
  • Balance

11. Accessibility in design ensures:

  • Usability for people with disabilities
  • Faster downloads
  • Colorful UI
  • Limited content

12. What is usability testing?

  • Testing colors only
  • Checking source code
  • Evaluating marketing strategy
  • Observing real users interacting with the product

13. Which layout principle helps guide the user’s eye smoothly across the page?

  • Texture
  • Alignment
  • Visual hierarchy
  • Typography

14. Which typography rule improves readability?

  • Using many fonts
  • Using consistent font size and spacing
  • Bold all text
  • Use uppercase everywhere

15. Empathy mapping in UX helps designers:

  • Track metrics
  • Create color palettes
  • Manage deadlines
  • Understand users’ feelings, thoughts, and needs
